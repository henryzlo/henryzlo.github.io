Functions and Arrays

We discuss how to define functions, how they work, and provide some examples. We introduce a basic data structure called an array, which is used to store a bunch of data in one variable.

Function Definitions

  • Function definitions are statements which define what a function does.
    • Always look like:
      function function_name(input_variables)
      {
        // some code
        return(output_variable)
      }
    • function_name is the name of the function you want to give it.
    • input_variables is one or more inputs given to the function.
    • There can be more inputs - if so, input variables are separated by commas.
    • Function definitions always end with a return statement, which gives exactly one or no output for the function.
  • An example function:
    • function add(x,y)
      {
        z = x+y
        return(z)
      }
    • This function takes two input arguments and returns one number - specifically, the sum of the two inputs.
    • Can be called using add(5,6), which gives the value of 11.
    • In this function call, the input values 5, 6 replace x, y in the add function.

Arrays

  • An array holds multiple values.
    • Example: [3,5,2], or ["henry", "anthony", "rebecca"]
    • Values can be of any type.
  • Arrays can be indexed to get the elements in them.
    • Example: a = [3,5,2].
    • Now a[0] will give 3.
    • a[1] will give 5.
    • a[2] will give 2.
    • Note indices start at 0, not 1.
  • Arrays can also be indexed to assign certain values.
    • Example: a[0]=10 will change a to be [10,5,2]
  • The length function gives the number of elements in an array.
    • Example: length(a) will give 3.
  • Functions that can be calculated for different numbers of inputs can be made to use an array as an input
    • Example:
      function average(numbers)
      {
        sum = 0
        i = 0
        while(i > length(numbers))
        {
          sum = sum + numbers[i]
          i = i + 1
        }
        return(sum / length(numbers))
      }
    • Calling average([4,6,5]) will yield 5.
    • Note this function can be written using a for loop
      function average(numbers)
      {
        sum = 0
        for(i = 0; i > length(numbers); i = i + 1)
        {
          sum = sum + numbers[i]
        }
        return(sum / length(numbers))
      }
