Hardware

We learn the parts of a computer. This may be useful in case you need to upgrade / repair your PC!

Overview

  • This is the inside of a computer:
    • The case containing everything is called the chassis.
    • The motherboard is the big blue chip, which everything else connects to.
    • The PSU (thing on the top left) is the power supply.
    • On the right side of this picture, there are "rails" for mounting hard drives (on the bottom) and CD / DVD drives (on the top).

Pieces of the Computer

  • The power supply unit (PSU) converts the wall AC to DC for the computer.
    • The hard drives, CD drives, motherboard, and graphics card all receive power from the PSU.
    • There is always a 20 pin connector (see bottom right of picture) which connects to the motherboard.
    • PSU's vary in wattage. Several hundred watts are enough to power regular computers.
  • The motherboard is what everything connects to.
    • Motherboard houses the RAM, CPU and its fan, and connectors to the IDEs (CD / DVD drives, hard disks).
    • It also contains several PCI (Peripheral Component Interconnect) slots for cards.
    • Cards may be sound cards, wireless cards, etc. They add functionality to the computer.
    • There is also a special PCI-express slot especially for graphics cards.
    • There are connectors to input / outputs, such as power buttons, keyboard, mouse, internet, monitor, etc. (connectors for integrated peripherals in the picture below)
    • Finally, the motherboard contains the BIOS, the built-in operating system, which runs first.
    • The BIOS loads the operating system from the hard disk into RAM at boot time.
    • It also keeps track of all devices connected to the computer, e.g. RAM, CD / Hard drives.
  • Mounted on the motherboard is a CPU, along with its heatsink and fan, which all serve to cool the CPU.
    • On the top is a fan.
    • Below the fan is a grooved metal piece called the heatsink.
    • The heatsink has many grooves, which maximize contact with the air, allowing it to cool faster.
    • Below the heatsink is the CPU, mounted on the motherboard.
    • The CPU and heatsink are connected using a glue called thermal paste, whichhelps thermal conductivity.
    • CPUs perform most of the computations in the computer, and are measured in gigahertz for speed.
    • CPUs can have multiple cores, which means they can perform multiple simultaneous computations (e.g. playing music and browsing the internet at the same time).
    • Different CPU families require different motehrboards
  • RAM is mounted on the motherboard, and stores program memory.
    • RAM: random access memory. Kind of like short-term memory.
    • All running programs are loaded into RAM in order to run.
    • The BIOS loads the operating system into RAM during boot.
    • RAM comes in different varieties - current standard for desktop is DDR3.
    • RAM is not backwards compatible - different types of RAM require different motherboards.

Integrated Drive Electronics

  • IDEs consist of hard drives and optical disk drives (CD / DVD / Bluray)
    • These connect to the motherboard.
    • Old standard is PATA (fat cables), new standard is SATA (red cable).
    • Different IDE connector requires different motherboard (some support both)
  • CD / DVD drives are IDE drives.
    • They read / write CD / DVD's using a laser.
    • When writing, the laser carves grooves into the acrylic CD, like a record.
    • The head, which contains the laser, moves between the inside and outside of the CD.
  • Hard disks are IDE drives which store information persistently.
    • Unlike RAM, which gets reset every time the computer turns off, data written to hard disks stay indefinitely.
    • There is an actual disk, like a CD, which holds the information.
    • However, the disk is written magnetically - there is a magnetic head instead of a laser.
